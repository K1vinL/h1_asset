HackerOne SRC 资产，每日更新

这个项目是一个 Python 脚本，用于批量更新 HackerOne 项目，获取 SRC 列表，并将 SRC 清单按项目存放到文件夹中。同时，使用 GitHub Action 每日自动运行脚本。

## 如何使用

### 克隆项目

克隆这个项目到本地：

```bash
git clone https://github.com/adysec/h1_asset.git
```

进入项目目录：

```bash
cd h1_asset
python h1_asset.py
```

HackerOne项目在 `src` 文件夹中
